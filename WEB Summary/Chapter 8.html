Chapter 8: Built-in Objects - Summary

1. Built-in Objects and the `window` Object

  • **Built-in Objects** are pre-created objects provided for the user to utilize in JavaScript.
  • The **`window` object** is the **top-level and fundamental object** in the JavaScript object hierarchy. All other JavaScript objects are contained within it.
  • The `window` object is used to control the state of the web browser window itself.
  • **Access**: You can access its properties and methods using dot notation (e.g., `window.alert()`), but `window.` is often optional (e.g., `alert()`).
  • **Key `window` Methods**:
    • `alert()`, `confirm()`, `prompt()`: Display dialog boxes.
    • `open()`: Opens a new browser window or tab. Syntax: `window.open(path, windowName, options)`.
    • `close()`: Closes the current window.
    • `resizeTo()`, `moveTo()`: Dynamically adjust the size or position of the window.

2. `screen` Object

The `screen` object holds information about the user's monitor screen size and orientation.

  • `screen.availWidth`, `screen.availHeight`: Returns the width/height of the available space on the screen, excluding persistent UI elements (like a taskbar). Used for centering popup windows.
  • `screen.width`, `screen.height`: Returns the total screen width/height, including UI elements.

3. `history` and `location` Objects

  • **`history` Object**: Stores the site addresses visited in the current browser window. It is read-only for security reasons.
    • `history.back()`: Navigates to the previous page in the history.
    • `history.forward()`: Navigates to the next page in the history.
    • `history.go(n)`: Navigates forward (`n > 0`) or backward (`n < 0`) by a relative number of pages.
  • **`location` Object**: Contains information related to the current document's URL.
    • `location.href`: The entire URL. Changing this value redirects the user to the new address.
    • `location.reload()`: Reloads the current document.
    • `location.protocol`, `location.host`, `location.pathname`: Specific parts of the URL.

4. `Date` Object

The `Date` object controls information about the current or specified date and time.

  • **Instance Creation**: To use the `Date` object, you must create an **instance** using the `new` keyword.
    Current Date: `let today = new Date();`
    Specific Date: `new Date("2025-02-25");` (YYYY-MM-DD format is common)
  • **Get Methods**: Used to retrieve time components (e.g., `getFullYear()`, `getMonth()` [0-11], `getDate()` [1-31], `getDay()` [0-6]).
  • **Set Methods**: Used to set time components (e.g., `setFullYear()`).
  • **Time Calculation**: The `getTime()` method returns the time since January 1, 1970, 00:00:00 UTC, in **milliseconds**. This is used to calculate the difference between two dates.
    Millisecond units: $1\text{s} = 1,000\text{ms}$, $1\text{day} = 86,400,000\text{ms}$.

5. `Math` Object

The `Math` object contains functions for mathematical operations (trigonometric, logarithmic, etc.). You do not need to create an instance to use it.

  • **Constants**: `Math.PI` (the value of Pi).
  • **Rounding Methods**:
    • `Math.round(x)`: Rounds to the nearest integer (standard rounding).
    • `Math.ceil(x)`: Rounds up to the smallest integer.
    • `Math.floor(x)`: Rounds down to the largest integer.
  • **Random Numbers**: `Math.random()` returns a random floating-point number between 0 (inclusive) and 1 (exclusive).
    To get a random integer between 1 and $N$: `Math.floor(Math.random() * N) + 1;`
  • **Number Formatting**: The `toFixed(n)` method (a method of the `Number` object) converts a number to a string, rounding to $n$ digits after the decimal point.
