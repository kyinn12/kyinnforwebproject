Chapter 10: Strings and Arrays - Summary

1. Accessing and Checking Strings (문자열 접근 및 확인)

  • **Autoboxing**: Primitive types (string, number, boolean) can use properties and methods because they are **temporarily converted** to their corresponding Wrapper Objects (`String`, `Number`, `Boolean`) when a method is called.
  • **Length**: Use the `.length` property: `string.length`.
  • **Character Access**:
    • Bracket notation (ES6+): `string[position]`.
    • `charAt()` method: `string.charAt(position)`.
  • **Position/Inclusion**:
    • `indexOf(substring, position)`: Returns the **first** starting index of the substring (or -1 if not found).
    • `includes(substring)` (ES6+): Returns `true` if the substring is found.
    • `startsWith(substring)`: Checks if the string begins with the substring (case-sensitive).
    • `endsWith(substring)`: Checks if the string ends with the substring (case-sensitive).

2. Modifying Strings (문자열 수정)

String methods do not modify the original string; they return a new string.

  • **Whitespace Removal**: Removes common whitespace (space, tab, newline).
    • `string.trim()`: Removes spaces from **both ends**.
    • `string.trimStart()`: Removes spaces from the **start**.
    • `string.trimEnd()`: Removes spaces from the **end**.
  • **Case Conversion**:
    • `string.toUpperCase()`: Converts all letters to **uppercase**.
    • `string.toLowerCase()`: Converts all letters to **lowercase**.
  • **Substring Extraction**: Extracts a portion of the string.
    • `string.substring(start, end_before)`: Extracts from the start index up to (but not including) the end index.
    • `string.slice(start, end_before)`: Same as `substring()`, but **allows negative indices** (counting backward from the end).
  • **Splitting**: `string.split(delimiter)`: Splits a string into an **array** of substrings based on a delimiter (e.g., splitting an email by `@`).

3. Regular Expressions (정규 표현식)

  • A powerful language for defining and searching for **specific patterns** (rules) in a string.
  • **Basic Structure**: `pattern[flags]`.
  • **Flags (Options)**:
    • `i`: Case-**insensitive** search.
    • `g`: **Global** search (finds all matches, not just the first).
  • **RegExp Methods**:
    • `regex.test(string)`: Returns `true` if a match is found, `false` otherwise.
    • `regex.exec(string)`: Returns the match as an array or `null`.
  • **String Methods using RegEx**: `string.match(regex)`, `string.replace(regex, replacement)`.
  • **Pattern Cheatsheet**:
    • `\d`: Digit (0-9). `\D`: Non-digit.
    • `\s`: Whitespace. `\S`: Non-whitespace.
    • `\w`: Word character (letters, digits, underscore). `\W`: Non-word character.
    • `^`: Start of the string (or NOT if inside `[]`).
    • `$`: End of the string.
    • `{n}`: Repeats pattern exactly `n` times. `pattern{n,}`: Repeats at least `n` times.

4. String and Array Conversion

  • **Why Convert?**: Arrays have many methods for modifying elements that strings do not have. Strings are often converted to arrays to be manipulated, then converted back.
  • **String to Array**:
    • `string.split("")`: Splits into an array of individual characters.
    • Spread Operator: `[...string]`.
    • `Array.from(string)`.
  • **Array to String**:
    • `array.join(delimiter)`: Joins all elements into a single string using the specified delimiter (e.g., `array.join("")` for no space).

5. Using Arrays Efficiently (배열 활용)

  • **Creating Arrays**: Use literal notation (`let arr = ["a", "b"]`) or the `Array` constructor.
  • **Iteration (Looping)**:
    • `for` loop (allows `break`).
    • `forEach()` (must iterate through the entire array; cannot `break`).
      Syntax: `array.forEach((value, index, array) => { ... });`
  • **Combining**:
    • `array1.concat(array2, value)`: Creates a new array by concatenating existing arrays/values.
    • **Spread Operator (ES6+)**: `[...array1, ...array2]` (Commonly used).
  • **Sorting/Reversing**:
    • `array.reverse()`: Reverses the order of elements (in place).
    • `array.sort()`: Sorts elements (requires a separate comparison function for numbers).
  • **Adding/Removing from Ends**:
    • **End**: `array.push(value)` (adds), `array.pop()` (removes last).
    • **Start**: `array.unshift(value)` (adds), `array.shift()` (removes first).
      *(Note: `shift`/`unshift` can be slower than `push`/`pop` for large arrays as indices must be renumbered.)*
  • **Adding/Removing Anywhere**:
    • **`array.splice(start, count, item1, ...)`**: Deletes `count` elements starting at `start`, and optionally inserts new items. **Modifies the original array**.
    • **`array.slice(start, end_before)`**: Extracts a portion into a new array. **Does not modify the original array**.
