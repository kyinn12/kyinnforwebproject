Chapter 14: Drawing Graphics with Canvas - Summary

1. Introduction to Canvas

  • The **`<canvas>` tag** provides an area in the web document for drawing graphics, often referred to as the 'canvas'.
  • The **Canvas API**, integrated into HTML, allows drawing graphics directly onto the document using JavaScript.
  • Canvas draws graphics pixel by pixel, allowing for fine-grained control, making it ideal for **games** or detailed effects rather than simple UI or charts.
  • The **origin (0, 0)** point for canvas coordinates is the **top-left corner**. X-coordinates increase to the right, and Y-coordinates increase downward.
  • Angles in Canvas API methods are expressed in **radians** ($ \text{radians} = (\pi/180) \times \text{degrees} $).

2. Creating the Rendering Context

  • The `<canvas>` tag only creates the area. To start drawing, you must create a **rendering context**.
  • Use the `getContext()` method:
    const ctx = canvas.getContext("2d"); (For 2D graphics).
  • The `ctx` object holds all the drawing properties and methods.

3. Drawing Basic Shapes (사각형 및 경로)

Rectangle Methods (기본 도형)

The rectangle is the only shape built directly into the Canvas API.

  • `ctx.fillRect(x, y, width, height)`: Draws a **filled** rectangle.
  • `ctx.strokeRect(x, y, width, height)`: Draws a rectangle with only a **border**.
  • `ctx.clearRect(x, y, width, height)`: Clears a rectangular area.

Path Methods (경로를 이용한 도형)

All other shapes (triangles, circles, curves) must be drawn using paths.

  • `ctx.beginPath()`: Starts a new drawing path.
  • `ctx.moveTo(x, y)`: Moves the starting point of the path to a new coordinate.
  • `ctx.lineTo(x, y)`: Draws a straight line path from the current position to the new coordinate.
  • `ctx.closePath()`: Closes the path by drawing a line back to the starting point (optional for filling).
  • `ctx.stroke()`: Draws the path outline (border).
  • `ctx.fill()`: Fills the area enclosed by the path with color.
  • **`Path2D` Object**: Used to store complex paths for reuse or repetition. `let path = new Path2D();`

Style Properties

  • `ctx.fillStyle = "color"`: Sets the color used for filling shapes (`fillRect`, `fill`).
  • `ctx.strokeStyle = "color"`: Sets the color used for borders/lines (`strokeRect`, `stroke`).
  • Styles apply to all subsequent drawing commands until explicitly changed.

4. Drawing Circles, Ellipses, and Curves

  • **Circle/Arc**: `ctx.arc(x, y, radius, startAngle, endAngle, counterclockwise)`
    • Angles are in radians. Full circle: `0` to `Math.PI * 2`.
    • `counterclockwise` (Boolean): determines the rotation direction (defaults to `true`).
  • **Ellipse**: `ctx.ellipse(x, y, radiusX, radiusY, rotation, startAngle, endAngle, counterclockwise)`
    • Requires separate horizontal (`radiusX`) and vertical (`radiusY`) radii.
    • `rotation`: Rotation amount in radians.
  • **Bezier Curves**: Draws smooth curves using control points.
    • **Quadratic (2nd order)**: One control point. `ctx.quadraticCurveTo(cpx, cpy, x, y)`.
    • **Bezier (3rd order)**: Two control points. `ctx.bezierCurveTo(cp1x, cp1y, cp2x, cp2y, x, y)`.
  • **Transformations**: The `ctx.scale(x, y)` method can be used to stretch or compress shapes (e.g., changing a circle into an ellipse).

5. Working with Images

  • **Loading an Image**:
    1. Create an Image object: `let img = new Image();`.
    2. Set the image source: `img.src = "filename.jpg";`.
    3. Use the `onload` event to ensure the image is loaded before drawing.
  • **Drawing Image**: Use the `drawImage()` method.
    • **Full Image, Default Size**: `ctx.drawImage(image, dx, dy)` (where to start drawing on canvas).
    • **Full Image, Resized**: `ctx.drawImage(image, dx, dy, dw, dh)` (draw width/height specified).
    • **Clipping/Slicing Source Image**: `ctx.drawImage(image, sx, sy, sw, sh, dx, dy, dw, dh)` (specifies source crop area `sx, sy, sw, sh` and destination area `dx, dy, dw, dh`).
  • **Image Clipping**: The `ctx.clip()` method hides any drawing outside the defined path, allowing images to be masked to a shape (e.g., a circle).
  • **Saving**: Graphics drawn on the canvas can be saved as an image file by right-clicking the canvas area in the browser and selecting "Save image as...".
