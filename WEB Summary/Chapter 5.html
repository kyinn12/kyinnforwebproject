Chapter 5: Introduction to DOM - Summary

1. Document Object Model (DOM)

  • The **Document Object Model (DOM)** is a standard that organizes a web document using objects, allowing **JavaScript** to access and control the web document systematically.
  • It makes the web page dynamically responsive to user actions or other conditions.
  • The DOM structures the web document into a **DOM Tree** (a hierarchical structure).
  • A **Node** is any item in the DOM tree where a branch splits off. The **root node** is always the `` element.
  • The DOM includes elements, their contents (text), and their attributes.

2. Accessing Web Elements

To modify or control an element, you must first "access" it.

  • **`querySelector(selector)`**: Returns the **first** element that matches the specified CSS selector.
  • **`querySelectorAll(selector)`**: Returns **all** matching elements as a **NodeList** (similar to an array).
  • **Legacy Access Methods (`getElement~`)**: Older methods that access elements by specific attributes:
    • `document.getElementById("idName")`.
    • `document.getElementsByClassName("className")`.
    • `document.getElementsByTagName("tagName")`.
  • **`querySelector()` Advantage**: It can combine multiple selectors (e.g., `#detail > p`), which the legacy methods cannot.

3. Getting and Modifying Element Content

These properties are used to get or set the text/HTML content of an accessed element.

  • **`element.innerText`**: Gets or sets the element's content as **plain text** (reflecting what is visible on screen).
  • **`element.innerHTML`**: Gets or sets the element's content, including **HTML tags**.
  • **`element.textContent`**: Gets or sets the element's content as raw text, including content from hidden elements and all whitespace found in the source.

4. Modifying Styles and Attributes

  • **Inline Style Modification**: Use the `.style` property to access and modify CSS properties.
    • CSS properties using hyphens (e.g., `background-color`) must be converted to **camelCase** in JavaScript (e.g., `style.backgroundColor`).
      title.style.backgroundColor = "yellow";
  • **Attribute Modification**: To change an element's attribute (like an image source), access the property directly.
    imageElement.src = "newImagePath.png";

5. Using `classList` Property

The `classList` property holds information about the class styles applied to an element.

  • **`element.classList.add("className")`**: Adds a new class style to the element.
  • **`element.classList.remove("className")`**: Removes an existing class style from the element.
  • **`element.classList.contains("className")`**: Checks if a specific class style is currently applied (returns `true` or `false`).
  • **`element.classList.toggle("className")`**: Toggles the class: if the class exists, it removes it; if it doesn't exist, it adds it.

6. Handling Form Elements

  • **Text Field Value**: To get the content a user entered into an input box, use the `.value` property.
  • **Access by Name Attribute**: If a `
    ` element has a `name` attribute, you can access form elements using dot notation:
    document.formName.elementName.value
  • **Access using Form Arrays**: All `` tags in a document are stored in the **`document.forms`** property (as an array/collection).
    Elements within a form are stored in the **`.elements`** property.
    document.forms[0].elements[4]
